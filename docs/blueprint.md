# **App Name**: Skyvera Listings

## Core Features:

- Property Listing Creation: Admin can add properties with title, description, price, location, property type, region (Dubai/UK), images, and status (Rent/Buy). Dubai properties also support 'Ready' and 'Off-plan' statuses.
- Property Filtering: Customers can filter properties by region, property type, location, status (Rent/Buy/Ready/Off-plan), and price range.
- Inquiry Submission: Customers can submit inquiries with name, email, phone, and message for specific properties.
- Inquiry Notification: Upon inquiry submission, an email with customer details and property link is sent to the admin.
- Valuation Request: Customers can request property valuations by submitting a form with their details and property link.
- Valuation Notification: An email notification is triggered to the admin upon a new valuation request, containing the customer's information and the property link.
- Admin Panel: Admin panel featuring login, property management (add/edit/delete), image upload, and views for customer inquiries and valuation requests.

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke trust and stability, aligning with real estate investment.
- Background color: Light gray (#F0F0F0), creating a clean and neutral backdrop for property listings.
- Accent color: Warm orange (#FFA500) for call-to-action buttons and important highlights.
- Body and headline font: 'Inter', a versatile sans-serif for clear readability and a modern feel.
- Use modern, geometric icons for property types, locations, and filters.
- Grid-based layout with clear spacing for easy browsing and comparison of properties.
- Subtle animations for transitions and loading states to enhance user experience.
